# Setup — Pongeek profile README

Quick setup guide for the synthwave/pixel-gaming profile README.

## 1. Create the magic repo

GitHub turns a repo named **exactly** the same as your username into your profile page.

1. Go to https://github.com/new
2. Repository name: `Pongeek` (must match your username exactly, case-sensitive)
3. Make it **Public**
4. Tick **Add a README file**
5. Click **Create repository**

## 2. Push these files

From this folder (`Pongeek-github-profile/`):

```bash
git init
git remote add origin https://github.com/Pongeek/Pongeek.git
git branch -M main
git add .
git commit -m "feat: synthwave pixel profile"
git push -u origin main --force
```

(The `--force` overwrites the default README GitHub auto-created in step 1.)

## 3. Turn on the snake animation

The contribution-graph snake is generated by a GitHub Action on a cron.

1. Visit `https://github.com/Pongeek/Pongeek/actions`
2. If prompted, click **I understand my workflows, go ahead and enable them**
3. Click **Generate Snake** in the left sidebar → **Run workflow** → **Run workflow** (one-time kick to populate the `output` branch)
4. After ~30 sec, refresh your profile — the snake should appear

It will then auto-refresh every 24h.

## 4. Final touches (edit `README.md`)

Replace the placeholders before you push:

- `mailto:your-email@example.com` → your real email (or remove the badge)
- `https://linkedin.com/in/your-handle` → your real LinkedIn URL (or remove the badge)
- **StudyQuest pin card** — will show "Repository not found" until you push StudyQuest to a public repo at `github.com/Pongeek/StudyQuest`. Until then either:
  - push StudyQuest publicly, OR
  - delete that one `<a href=".../StudyQuest">...</a>` block from `current_quests.exe`

## 5. (Optional) Pin the right repos on your profile

The big tiles on your profile page are separate from the README cards.

1. Go to https://github.com/Pongeek
2. Click **Customize your pins**
3. Pick StudyQuest / Portfolio / TileTech / CoupCoupon-client

## What you'll get

- **Animated waving header** with your name in pixel font
- **Typewriter tagline**: "Building AI that makes learning feel like a game"
- **Live stats cards** (commits, streak, top languages) in synthwave theme that update automatically
- **RPG character sheet** vibe for your tech stack
- **Project pin cards** themed to match
- **Trophy case** showing your GitHub achievements
- **Animated snake** eating your contribution graph in neon pink

## Tweaking colors later

The palette is:

| role         | hex       |
|--------------|-----------|
| background   | `#0D0221` |
| neon pink    | `#FF2A6D` |
| neon cyan    | `#05D9E8` |
| purple mid   | `#6A1B9A` |
| text         | `#D1F7FF` |

Find-and-replace these in `README.md` if you ever want to re-skin (e.g. to GameBoy green, swap `#FF2A6D` → `#9BBC0F`, `#05D9E8` → `#306230`, `#0D0221` → `#0F380F`).
